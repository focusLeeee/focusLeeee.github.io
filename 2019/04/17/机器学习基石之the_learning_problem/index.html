1. what is machine learning

  • learning: 通过观察学习skills技巧
    skills: improve some performance measure(e.g. prediction accuracy)
  • machine learning：通过对数据进行学习/计算获得skills
  • key essence of machine learning
    • exists some ‘underlying pattern’ to be learned有一些潜藏的模式
      • 所以可以提高一些性能度量performance measure
    • but no programmable(easy) definition(所以需要ML)
    • somehow there is data about the pattern（所以有可以学习的输入）

3. components of machine learning

这一小节是第一章的重点，因为它将机器学习的理论应用符号及数学知识进行表示，而以下各章内容也都是在这小节内容的基础上展开的。

  • 从一个银行是否会发信用卡给用户的例子引出了机器学习可以分为哪几个部分（组件）。
    • 输入(input)：$x \in X$（代表银行所掌握的用户信息）
    • 输出(output)：$y \in Y$ （是否会发信用卡给用户）
    • 未知的函数，即目标函数（target function）：$f: X \rightarrow Y$理想的信用卡发放公式）
    • 数据或者叫做资料（ data），即训练样本（ training examples）：$D=\{(x_1, y_1), (x_2, y_2), …, (x_n, y_n)\}$（银行的历史记录）
    • 假设（hypothesis），即前面提到的技能，能够具有更好地表现：$g: X \rightarrow Y$（能够学习到的公式）

4. machine learning and other fields

4.1 machine learning vs. data mining

机器学习与数据挖掘者叫知识发现（KDD Knowledge Discovery in Dataset）

  • data mining: 从大量的数据中找出有用的信息
  • 一直点：能够找出的有用信息就是我们要求得的近似目标函数的假设
  • 互助：能够找出的有用信息就能帮助我们找出近似的假设，反之也可行。
  • 传统的数据挖掘更关注与从大量的数据中的计算问题

4.2 machine learning vs. artificial intelligence

  • 人工智能的大概概念就是电脑能够表现出一些智慧行为。
  • 机器学习是实现人工智能的一种方式。

4.3 machine learning vs. statistic

  • 统计也需要通过数据，来做一个未知的推论。
  • 统计是一种实现机器学习的方法。
  • 传统的统计学习关注于数学公式，而非计算本身。focus on provable results with math assumptions, and
  • c
  • 机器学习use data to compute hypothesis $g$ that approximates target f
  • statistics use data to make inference about un unknown process
